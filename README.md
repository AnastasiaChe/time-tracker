# Anastasia Che Time Tracker

Персональный трекер времени для фрилансера: клиенты, проекты, таймер, отчеты, импорт Clockify PDF и экспорт PDF.

## Запуск

```bash
/Users/anastasiache/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 app.py 8000
```

Открыть:

```text
http://127.0.0.1:8000
```

## Что есть

- Клиенты: название, контакт, email, валюта, удаление с подтверждением.
- Проекты: клиент, название, ставка, валюта, удаление с подтверждением.
- Таймер: старт/стоп из шапки, заполнение деталей во время работы, повторный старт по существующей записи.
- Отчеты: фильтры по датам, готовым периодам, клиенту, проекту и тегам.
- Time entries: редактирование, удаление, сумма к оплате, индикация перехода на следующий день.
- Импорт PDF: парсит приложенный Clockify Summary и создает записи из агрегированных строк проектов.
- Экспорт PDF: строит отчет по текущим фильтрам.
- Иконки: локально подключен Font Awesome Free из `static/vendor/fontawesome`.

Данные лежат в `data/time_tracker.sqlite3`.
