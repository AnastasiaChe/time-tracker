# Сторонние компоненты

В проект включены сторонние открытые компоненты, чтобы приложение работало локально без CDN и без дополнительных установок.

## Google Fonts

Файлы шрифтов лежат в `static/vendor/fonts/`.

- Roboto: https://fonts.google.com/specimen/Roboto
- Mulish: https://fonts.google.com/specimen/Mulish

Шрифты распространяются Google Fonts на условиях их открытых лицензий. Эти лицензии применяются только к файлам шрифтов и не меняют лицензию самого приложения.

## Font Awesome Free

Файлы Font Awesome лежат в:

- `static/vendor/fontawesome/`
- `chrome-extension/vendor/fontawesome/`

Font Awesome Free распространяется на условиях своей открытой лицензии. Эта лицензия применяется только к файлам Font Awesome и не меняет лицензию самого приложения.
